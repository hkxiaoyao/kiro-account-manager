✨ 为什么选择 Kiro Gateway？

🚀

高性能 Rust 实现

基于 Rust + Axum 构建，内存安全、并发高效。相比 Python/Node.js 实现，性能提升 10 倍以上，资源占用更低。

🔄

智能账号管理

多账号轮询、自动 Token 刷新、配额监控、限流跳过、过期标记。无需手动管理，全自动运行。

🔌

完整 API 兼容

支持 OpenAI Chat Completions API 和 Anthropic Messages API，无缝对接现有应用，无需修改代码。

⚡

流式响应 + 工具调用

完整支持流式响应（SSE）、工具调用、图片上传、Thinking block 解析，功能与官方 API 一致。

💡 使用场景

🤖 AI 应用开发

使用 OpenAI SDK 或 Anthropic SDK 开发 AI 应用，只需修改 base_url 即可接入 Kiro API，无需修改代码逻辑。

# Python 示例
client = OpenAI(
  base_url="http://localhost:8080/v1"
)

🔌 第三方工具集成

将 Kiro API 接入到支持 OpenAI API 的第三方工具，如 ChatBox、OpenCat、BotGem 等。

✓ ChatBox、OpenCat 等桌面客户端
✓ BotGem、NextChat 等 Web 应用
✓ Cursor、Continue 等编程工具
